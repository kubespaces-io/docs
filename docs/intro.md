# Introduction

Let's discover **Kubespaces in less than 5 minutes**.

## Getting Started

Get started by **signing up on [kubespaces.io](https://app.kubespaces.io)**.


### What you'll need

- A web browser
- If you want to interact with the Kubernetes cluster, you'll need to install the [kubectl](https://kubernetes.io/docs/tasks/tools/) command-line tool.
- You can use Kubespaces also using the [Kubespaces CLI](https://docs.kubespaces.io/cli/overview.html).

## What is Kubespaces?

Kubespaces is a Kubernetes-native platform for building and deploying applications on Kubernetes. It is designed to be a simple and easy-to-use platform for developers and operators to build, deploy, and manage applications on Kubernetes.
